# virtual-herbal-garden
Virtual Herbal Garden is a web-based interactive platform that digitally showcases medicinal plants with detailed information about their scientific names, uses, health benefits, and traditional importance. The project aims to spread awareness about herbal medicine through an engaging, user-friendly, and responsive interface.
